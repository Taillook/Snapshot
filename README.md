# Installation
`pip3 install git+https://github.com/Taillook/Snapshot`

# Usage
```
$ snapshot
took a picture at 'file_name'
```

```
$ snapshot -d 'delay_time(int)'
took a picture at 'file_name'
```

```
$ snapshot -p 'file_name(str)'
took a picture at 'file_name'
```

```
$ snapshot -o 'file_name(str)'
took a picture at 'file_name'

and open image(mac only)
```
